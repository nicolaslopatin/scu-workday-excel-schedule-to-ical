SCU Workday Excel Schedule to iCal

About This Tool

This tool creates calendar events for your SCU classes, by generating an iCal (.ics) file from an exported Excel spreadsheet of your SCU schedule from Workday.

This tool was created by Nicolas Lopatin, a student at Santa Clara University. It is not affiliated with or endorsed by Santa Clara University or Workday.

How To Use
  1. Log in to Workday.
  2. Navigate to your class schedule (Academics > View My Courses) or alternately search for classes in a specific academic period (Academics > SCU Find Course Sections).
  3. Press the "Export to Excel" button above the table (on the right) to download your schedule as an Excel file.
  4. Select the exported Excel file using the file input below.
  5. Specify your desired alert times (in minutes before each event) using the text field below.
  6. Click "Generate iCal" to download the .ics file.
  7. Import the .ics file to your preferred calendar application (e.g., Google Calendar, Apple Calendar, Outlook) and enjoy!

Note: This tool processes the Excel file entirely in your browser, and no data is sent to any server. All of your information stays on your device.
